# discord-bot

bot developed in typescript

## How do you use it?

you must create a file to define the environment variables

Create `.env` file:
```env
TOKEN=TOKEN
PREFIX=PREFIX
```

To compile, simply run the command:
```bash
npm run build
```

## Testing

To test, simply run the command:
```bash
npm run start
```
